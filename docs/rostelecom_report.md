# Отчет о взаимодействии с организацией-партнером

**Студент:** Дмитриев Кирилл Денисович
**Партнер:** ПАО «Ростелеком»
**Формат взаимодействия:** Очное мероприятие на базе университета.

## Описание мероприятия
В рамках проектной практики я принял участие в профильном мероприятии, организованном представителями ПАО «Ростелеком». На мероприятии обсуждались современные IT-решения и ИИ технологии.

## Полученные знания
В ходе мероприятия я узнал:
1. Как организуются серверные мощности для поддержки высоконагруженных систем.
2. Какие решения «Ростелеком» предлагает для разработчиков программного обеспечения.
3. Уникальные возможности современных ИИ технологий.

## Связь с проектом "Forest Ranger"
Мой проект представляет собой мобильную игру по модели Pay-to-play. Несмотря на то, что это одиночная игра, в перспективе планируется внедрение **Системы облачных сохранений профиля игрока**.

Знания, полученные на мероприятии «Ростелекома», дали мне понимание того, как работают облачные инфраструктуры и какие серверные решения можно использовать для реализации сетевого функционала игры при релизе на таких платформах, как RuStore. Взаимодействие с партнером позволило расширить технический кругозор и применить системный подход к проектированию игры.
